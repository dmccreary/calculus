## About This MicroSim

This calculator walks through the FTC Part 2 evaluation process:

## Lesson Plan

### Learning Objectives

Students will be able to:

1. Apply FTC Part 2 to evaluate definite integrals
2. Follow the standard evaluation procedure: find antiderivative, apply limits, subtract
3. Use the bracket notation $[F(x)]_a^b$ correctly

### Activities

1. **Step-Through Practice (10 min)**: Work through each problem, predicting each step
2. **Connection to Area (5 min)**: Observe how the numerical answer matches the shaded area
3. **Speed Drill (5 min)**: Try to complete problems faster with each iteration

### Evaluation Procedure

1. Write the integral: $\int_a^b f(x)\,dx$
2. Find antiderivative: $F(x)$
3. Apply bracket notation: $[F(x)]_a^b$
4. Evaluate: $F(b) - F(a)$
5. Simplify to final answer
