# References: Basic Derivative Rules

1. [Differentiation rules](https://en.wikipedia.org/wiki/Differentiation_rules) - Wikipedia - Complete reference for all standard derivative rules including power, constant multiple, and sum/difference rules with proofs. Directly covers every rule introduced in this chapter.

2. [Power rule](https://en.wikipedia.org/wiki/Power_rule) - Wikipedia - Detailed proof and applications of the power rule for integer and rational exponents. The fundamental rule that forms the backbone of this chapter.

3. [Linearity of differentiation](https://en.wikipedia.org/wiki/Linearity_of_differentiation) - Wikipedia - Explains why the derivative of a sum equals the sum of derivatives and why constants factor out, with formal proofs of these essential properties.

4. Calculus: Early Transcendentals (9th Edition) - James Stewart - Cengage Learning - Section 3.1 develops the power rule, constant multiple rule, and sum rule with applications to polynomial and root functions.

5. Thomas' Calculus (15th Edition) - Joel Hass, Christopher Heil, Maurice Weir - Pearson - Section 3.3 provides systematic coverage of basic differentiation rules with numerous examples organized by increasing complexity.

6. [Differentiation Formulas](https://tutorial.math.lamar.edu/Classes/CalcI/DiffFormulas.aspx) - Paul's Online Math Notes - Concise reference sheet with all basic derivative formulas and worked examples applying each rule to various function types.

7. [Basic Derivative Rules](https://www.khanacademy.org/math/ap-calculus-ab/ab-differentiation-1-new/ab-2-6a) - Khan Academy - Practice exercises on applying the power rule, constant rule, and sum/difference rule with step-by-step solutions.

8. [Power Rule Proof and Examples](https://www.youtube.com/watch?v=EY6FHX6aSkg) - Organic Chemistry Tutor - Comprehensive video covering the power rule for positive, negative, and fractional exponents with extensive worked examples.

9. [Derivative Rules](https://www.mathsisfun.com/calculus/derivatives-rules.html) - Math is Fun - Clean visual presentation of basic derivative rules with interactive examples, ideal for quick reference and practice.

10. [Derivatives of Polynomials and Exponentials](https://www.whitman.edu/mathematics/calculus_online/section03.02.html) - Whitman College Calculus - Open-source textbook section with proofs, examples, and exercises for the core differentiation rules.
